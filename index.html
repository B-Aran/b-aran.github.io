Khemia

Khemia is the result of a final project master thesis developed by 40 people of different departments with a variety of backgrounds: design, art, sound, programming, production... As a member of the programming team I have conrtibuted to adding new features using C++ with Unreal Engine 4 and maintaining them, as well as solving bugs. Developing puzzle map features and managing the main logic of the game which is the teleportation between different rooms, integrating FMOD and sounds to the game and developing the main menus UIs are examples of my tasks in this project.

Main Technologies: UE4, C++, Perforce, FMOD and HackNPlan.

Building a computer

Building a computer from scratch, starting with only a chip representing the NAND logical door. The main goal is to achieve a basic Tetris program, following nand2tetris course. Includes main chips, an ALU, memory and assembler, a basic operating system and more!

Main Technologies: HDL (Hardware Description Language), Hardware Simulator, ASM
