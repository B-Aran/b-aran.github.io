Khemia

Khemia is the result of a final project master thesis developed by 40 people of different departments with a variety backgrounds: design, art, sound, programming, production... As a member of the programming team I have conrtibuted to adding new features using C++ with Unreal Engine 4 and maintaining them, as well as solving bugs. Developing puzzle map features and managing the main logic of the game which is the teleportation between different rooms, integrating FMOD and sounds to the game and developing the main menus UIs are examples of my tasks in this project.

Main Technologies: UE4, C++, Perforce, FMOD and HackNPlan.

Chronicles of Elathudel

Personal project inspired by Brackeys and Sebastian Lague where the intention is to make a basic RPG game with procedural terrain generation. In the RPG part we can find concepts such as movement, camera, the inventory mechanic, UI, combat and AI. In relation with the procedural terrain generation we can find different level of detail depending how far the terrain chunk is, noise maps and generation of textures and meshes from created material and tools in order to make easier the creation of maps.

Main Technologies: Unity and C#.
